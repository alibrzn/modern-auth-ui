## ⚡ Installation

Clone Repository

```bash
git clone https://github.com/alibrzn/modern-auth-ui.git
```

Move to Project Folder

```bash
cd login-sign
```

Install Dependencies

```bash
npm install
```

Run Development Server

```bash
npm run dev
```
